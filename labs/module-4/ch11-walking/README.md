# Lab 11: Humanoid Locomotion

## Overview

Implement balance control and walking gaits for bipedal robots.

## Prerequisites

- Completed Module 3
- Strong dynamics background

## Learning Outcomes

- Implement ZMP balance
- Generate walking gaits
- Handle perturbations
- Integrate with ros2_control

## Files

- `balance_controller.py` - Balance implementation
- `gait_generator.py` - Walking patterns
- `INSTRUCTIONS.md` - Step-by-step guide

## Expected Duration

3-4 hours
