# Lab 12: Dexterous Manipulation

Implement grasp planning and force control for robotic hands.

## Prerequisites

- Completed Lab 11

## Learning Outcomes

- Model hand kinematics

- Implement force control
- Handle contact dynamics

- `force_controller.py` - Force control
- `INSTRUCTIONS.md` - Step-by-step guide
