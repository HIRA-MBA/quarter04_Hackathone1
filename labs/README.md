# Lab Exercises

This directory contains hands-on lab exercises for each chapter of the Physical AI & Humanoid Robotics textbook.

## Structure

```
labs/
├── module-1/          # ROS 2 Fundamentals
│   ├── ch01-hello-robot/
│   ├── ch02-sensor-fusion/
│   ├── ch03-ros2-deep-dive/
│   ├── ch04-urdf-humanoid/
│   └── ch05-edge-controller/
├── module-2/          # Digital Twin
│   ├── ch06-gazebo-world/
│   └── ch07-unity-twin/
├── module-3/          # NVIDIA Isaac
│   ├── ch08-isaac-scene/
│   ├── ch09-gpu-vision/
│   └── ch10-sim2real/
└── module-4/          # VLA & Capstone
    ├── ch11-walking/
    ├── ch12-manipulation/
    ├── ch13-vla-commands/
    └── ch14-capstone/
```

## Getting Started

1. Complete the hardware and software setup in the textbook's Front Matter section
2. Navigate to the lab directory for your current chapter
3. Follow the INSTRUCTIONS.md file in each lab
4. Refer back to the chapter for theoretical background

## Requirements

- ROS 2 Humble
- Ubuntu 22.04 or compatible Linux distribution
- See individual lab README files for specific requirements
