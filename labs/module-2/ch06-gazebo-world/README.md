# Lab 6: Gazebo & Physics Simulation

## Overview

Create a Gazebo world and spawn your robot model.

## Prerequisites

- Completed Module 1
- Gazebo Harmonic installed

## Learning Outcomes

- Create SDF world files
- Configure physics parameters
- Spawn robots programmatically
- Bridge Gazebo-ROS 2 topics

## Files

- `robot_world.sdf` - Gazebo world file
- `spawn_robot.py` - Spawning script
- `INSTRUCTIONS.md` - Step-by-step guide

## Expected Duration

60-90 minutes
