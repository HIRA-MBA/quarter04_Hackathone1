# Lab 7: Unity Digital Twin Capstone

## Overview

Build a high-fidelity digital twin using Unity and ROS 2.

## Prerequisites

- Completed Lab 6
- Unity 2022 LTS
- Basic C# knowledge

## Learning Outcomes

- Set up Unity Robotics Hub
- Connect Unity to ROS 2
- Synchronize twin state
- Create visualization dashboards

## Files

- `RosBridge.cs` - Unity ROS connector
- `INSTRUCTIONS.md` - Step-by-step guide

## Expected Duration

2-3 hours
