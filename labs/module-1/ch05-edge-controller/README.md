# Lab 5: Edge Deployment Capstone

## Overview

Deploy ROS 2 applications to edge devices (Jetson, Raspberry Pi).

## Prerequisites

- Completed Labs 1-4
- Edge hardware available
- Docker installed

## Learning Outcomes

- Cross-compile ROS 2 packages
- Optimize for edge devices
- Deploy via containers
- Monitor edge performance

## Files

- `edge_node.py` - Optimized edge node
- `Dockerfile` - Container configuration
- `INSTRUCTIONS.md` - Deployment guide

## Expected Duration

2-3 hours
