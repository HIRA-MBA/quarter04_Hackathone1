# Lab 4: URDF & Humanoid Modeling

## Overview

Build a humanoid robot description using URDF and Xacro.

## Prerequisites

- Completed Labs 1-3
- Basic 3D modeling concepts

## Learning Outcomes

- Write URDF robot descriptions
- Use Xacro for modular definitions
- Add visual and collision meshes
- Visualize models in RViz2

## Files

- `humanoid.urdf` - Basic URDF model
- `humanoid.xacro` - Xacro version
- `INSTRUCTIONS.md` - Step-by-step guide

## Expected Duration

90-120 minutes
