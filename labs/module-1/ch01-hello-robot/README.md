# Lab 1: Hello Robot - Your First ROS 2 Node

## Overview

In this lab, you will create your first ROS 2 node that publishes and subscribes to messages.

## Prerequisites

- ROS 2 Humble installed
- Python 3.10+
- Completed Chapter 1 reading

## Learning Outcomes

- Create a ROS 2 Python package
- Implement a publisher node
- Implement a subscriber node
- Use ROS 2 CLI tools for debugging

## Files

- `hello_node.py` - Your first ROS 2 node (to be created)
- `package.xml` - ROS 2 package manifest
- `setup.py` - Python package configuration
- `INSTRUCTIONS.md` - Step-by-step lab guide

## Expected Duration

45-60 minutes
