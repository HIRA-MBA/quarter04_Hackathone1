# Lab 3: ROS 2 Architecture Deep Dive

## Overview

Implement advanced ROS 2 patterns: lifecycle nodes, services, and actions.

## Prerequisites

- Completed Labs 1-2
- Understanding of async programming

## Learning Outcomes

- Create managed lifecycle nodes
- Implement custom services
- Build action servers and clients
- Handle parameters dynamically

## Files

- `lifecycle_node.py` - Lifecycle node example
- `service_node.py` - Service implementation
- `action_node.py` - Action server/client
- `INSTRUCTIONS.md` - Step-by-step guide

## Expected Duration

90-120 minutes
