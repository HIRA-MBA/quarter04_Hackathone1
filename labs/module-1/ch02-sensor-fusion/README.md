# Lab 2: Sensor Fusion

## Overview

Process and fuse data from multiple sensors including camera, LIDAR, and IMU.

## Prerequisites

- Completed Lab 1
- Sensor hardware or Gazebo simulation
- OpenCV installed

## Learning Outcomes

- Subscribe to sensor topics
- Process image data with cv_bridge
- Fuse multi-modal sensor data
- Visualize results in RViz2

## Files

- `sensor_node.py` - Sensor processing node
- `INSTRUCTIONS.md` - Step-by-step guide

## Expected Duration

60-90 minutes
