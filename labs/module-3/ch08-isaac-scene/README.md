# Lab 8: Isaac Sim Fundamentals

## Overview

Create your first scene in NVIDIA Isaac Sim.

## Prerequisites

- Completed Module 2
- Isaac Sim installed
- RTX GPU with 8GB+ VRAM

## Learning Outcomes

- Navigate Isaac Sim UI
- Create USD scenes
- Import robot assets
- Write Python extensions

## Files

- `scene.usd` - Isaac Sim scene
- `spawn_robot.py` - Python script
- `INSTRUCTIONS.md` - Step-by-step guide

## Expected Duration

60-90 minutes
